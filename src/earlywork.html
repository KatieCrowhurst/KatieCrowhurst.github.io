Learning JavaScript

I taught myself JavaScript through Udemy videos in my spare time after work. The course I took included theory, as well as practical projects to help solidify my learning. You can see some of the work I completed as part of this course below.

Dice Game

The first practical project on the course was a dice game. Players could input a target number to reach and players would take turns to roll the dice. The game was based on the dice game "Pig", and so had to incorporate certain rules, i.e. if a player rolls a one they lose all of their points accumulated for that round.
This was my first time using JavaScript and I learned how to use event listeners (to listen for the click event on the roll/hold buttons), how to use if statements (to put in place the rules, and to identify if the target score had been met), and some of JavaScript's math functions (Math.random to generate a random number).

Budget Calculator

The budget calculator app as you can see in this video, allows for user input. A user can add their monthly income/expenses and the app will display these in two separate lists. The app will also work out what percentage each expense is of the total income.
In this project I was introduced to the concept of controllers to separate and order my code, shown how to use and manipulate lists, had to carry out more mathematical equations (to work out the percentages), and how to interact with my HTML elements (for example how to update my UI elements smoothly with each new addition/subtraction/percentage change).

Forkify App

The Forkify app was my first time interacting with an API. A user is able to search for a food item, and the app will return a list of related recipes. The user can then view the recipe, and even add the ingredients to a shopping list.
I split my code into separate views and models so that I could keep related code/functions together, to make it easier to understand. Prior to this the other projects had been using JavaScript version ES5, but this project allowed me to explore version ES6.
This was also the first time I had been introduced to async/await. To handle loading of recipes in the UI once a search query had been initiated, I was able to render a loading spinner to visually indicate to a user that their request is currently being handled.
