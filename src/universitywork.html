This was an assessment from one of my favourite modules at University. Here we were tasked with using HTML, CSS, JavaScript, PHP and MySQL to create a theatre booking website. This module was my first experience with PHP, and it allowed me to also look at MySQL more in depth than I had previously.
For this task we had to display performances to our users, allow a user to pick a performance, pick a seat, and receive a booking receipt at the end of the process. Unfortunately at the time I could not finish this assessment in time as I could not quite get the booking details to display correctly on the final page. Whilst I ended up submitting an incomplete assessment, I learned a lot from creating the website. 5 months after this assessment, I actually then came up against the same issue with my PHP code during my dissertation project, but this time I was successful in fixing the issue!
